getopt介绍

getopt 为Linux下处理命令行参数的函数，相对于自己编写代码来解析命令行参数， getopt提供了更为方便和友好的方式。

函数 getopt 包含于系统头文件 unistd.h 中，其函数原型为:

int getopt(int argc, char *const argv[], const char *optstring);
extern char *optarg;
extern int optind, opterr, optopt;
